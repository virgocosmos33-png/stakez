================================================================================
 RECONCEPT PIPELINE  —  Madam Mirror  (5x4 Ways slot on Stake Engine)
 A-Z checklist of EVERYTHING that was changed from the starter template
 so the same game can be re-skinned into a new concept end-to-end.
================================================================================

HOW TO USE THIS FILE
--------------------
This is the master reskin/reconcept map. It lists every layer that had to be
touched to turn the vanilla Stake `0_0_ways` template into "Madam Mirror".
When you reskin to a new concept, walk this file top-to-bottom and swap each
asset / config / string / audio cue for the new theme. Anything marked
[TEMPLATE-REMOVED] is dead template art we deleted — don't waste time on it.

Two independent halves must BOTH be reskinned and kept in lockstep:
  1. FRONTEND  -> web-sdk/apps/ways   (Svelte + PixiJS render layer)
  2. MATH      -> math-sdk/games/0_1_madam_mirror   (Python RGS math + books)
The frontend reads books/events produced by the math; if you rename symbols or
modes in one, you MUST mirror it in the other.

Legend:  [ASSET] art/audio file   [CODE] source file   [CFG] config/identity
         [GEN] generator/tool script   [TEMPLATE-REMOVED] deleted template art

--- GLASS / SHATTER FX SYSTEM  (the bespoke "ways-split" set-piece) ------------
[CODE]  src/components/MirrorShatter.svelte  — three distinct emitter events,
        all drawn live with PixiJS Graphics (no sprite for the shards):
   1) mirrorTelegraph  = the PRE-SPLIT VOLLEY. For each mirror->targets pair it
      fires 8 tiny glass SPIKES per target (SHARDS_PER_TARGET=8) that stab from
      the mirror cell into each target cell. Staggered rapid-fire, ease-in "stab
      punch", elongated spike silhouette pointed along flight, sharp hit-flash +
      violet core on arrival, plus pulsing target outlines. ~340ms window.
      >>> THIS is "the ones that shoot the mini small pieces of glass before the
          split". Tune: SHARDS_PER_TARGET, travel/delay, telegraphProgress dur.
   2) mirrorShatter    = the SPLIT BURST. 14 glass shards per cell explode
      outward with velocity + gravity + spin, fading over ~650ms (cubicOut).
      >>> THIS is "the animated thing we got for the split".
      Colors: 0xd8ecff (ice) / 0xb887ff (violet). Tune: count=14, duration 650.
   3) mirrorBreak      = swaps the cell to the cracked-glass overlay (glassBroken
      sprite) with a quick 130ms pop, ~420ms hold.
[ASSET] mirror/glass_intact.png, mirror/glass_broken.png  (assets.ts glassIntact/
        glassBroken) — the cracked-cell overlay art for mirrorBreak.
[AUDIO] sfx_xways_split (telegraph/split), sfx_mirror_break (shatter) in sounds.json.
[WIRING] mirrorTelegraph/mirrorShatter/mirrorBreak declared in
        src/game/typesEmitterEvent.ts + routed via bookEventHandlerMap.ts.

 R.  RESKIN ORDER OF OPERATIONS  (recommended sequence for next concept)
================================================================================
 1. Decide concept, rewrite GAME_CONCEPT + symbol meaning map (section B).
 2. Rename identity: config.ts + game_config.py + gameID + app/game folders (A).
 3. MATH first: reels, paytable, bet modes -> run.py -> books/publish (M).
 4. Symbols: spine rigs + static sheet + paytable icons (B).
 5. Backgrounds base+bonus (stills + ambient loops) (C).
 6. Character/mascot spine + poses (D).
 7. Frame + bottom HUD compartments + counters (E, F).
 8. Shared HUD: retheme accent/icons + control bar; rewrite gameInfo copy;
    verify social-casino text (T).
 9. Fonts (G).
10. Audio pack: edit assets-raw stems -> rebuild sprite: BGM/UI/SFX/celeb (H, V).
11. Bonus sequences: intro/outro/buy-menu/shatter FX + rocks->glass (I).
12. Win celebration films + tiers (J).
13. Intro/loading/logo/carousel + app shell (title/favicon/loader) (K, U).
14. i18n strings + localized plates (N).
15. (Art/video via AI: Scenario/Seedance generate -> post-process -> pack) (W).
16. QA sweep via Storybook + qa_*.py, update docs/compliance (O, P).
17. Build, test books, publish to Stake (M, Q).
